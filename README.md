# 🚀 DSA Tracker Website

This is a multi-page website created using HTML and CSS.  
It helps users track their daily DSA (Data Structures & Algorithms) practice and stay consistent.

---

## 🌟 Features

- 📌 Daily Problem Tracking  
- 🔥 Streak & Goals  
- 📊 Progress Overview  
- 👤 Signup & Login UI  
- 💬 Testimonials Section  

---

## 📂 Pages Included

- Home Page (index.html)  
- Features Page  
- Testimonials Page  
- Get Started Page  
- Signup Page  
- Login Page  
- Tracking Dashboard  

---

## 🛠️ Technologies Used

- HTML  
- CSS  

---

## ⚠️ Note

This is a front-end project.  
The forms (Signup/Login) are for UI demonstration only and do not store data.

This project was created as part of an assignment to practice:
